# 安全策略

## 支持范围

安全修复优先提供给当前最新 Release。旧版本可能不再单独修复，请先确认问题能否在最新版本复现。

## 私密报告漏洞

请通过 GitHub 的[私密漏洞报告](https://github.com/meyaomiao/wifi-health-console/security/advisories/new)提交安全或隐私问题。不要在公开 Issue、讨论、截图或日志中披露可利用细节、凭据、真实网络标识或个人数据。

报告中建议包含：

- 受影响的平台和应用版本
- 最小化且已脱敏的复现步骤
- 实际影响和可能的攻击条件
- 可选的修复建议

普通安装、使用和指标解释问题请使用公开的[使用问题模板](https://github.com/meyaomiao/wifi-health-console/issues/new?template=question.yml)。
